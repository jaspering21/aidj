# AIDJ App - B 阶段：用户留存 - 每日推荐

**日期**: 2026-05-02
**阶段**: B - 用户留存
**状态**: 已批准

---

## 设计目标

让用户每天都想打开 AIDJ，创造"每日专属推荐"的 habit。每次打开 app 都有新鲜感和期待感。

---

## 功能设计

### 1. 每日推荐触发

**触发时机**: 用户打开 app 时（每次）

**触发流程**:
1. 用户进入 app
2. AI 自动生成推荐（TTS 播报）
3. 播放推荐歌曲
4. 用户听完/跳过 → 显示"今日推荐已更新"

---

### 2. AI 播报内容

**首次推荐（A 阶段）**:
> "你好！我是 AIDJ，你的专属音乐 DJ 🎵 根据今天的天气，我为你选了一首很棒的歌——《{歌曲名}》，{艺术家}，希望你会喜欢 🎧"

**每日推荐（B 阶段，第二次及以后）**:
> "欢迎回来！根据你的品味，我为你准备了今日专属歌曲 🎵"

**播放完成后提示**:
> "今日推荐已更新 🎵 你可以继续听，或者告诉我你想听什么风格的音乐。"

---

### 3. 推荐逻辑优化

**A 阶段逻辑**: 天气 + 时间

**B 阶段新增**:
- 如果用户有播放历史，优先推荐与历史风格相似的歌曲
- 如果用户有收藏，优先从收藏风格中选择
- 如果是新用户，保持天气 + 时间逻辑

**优先级**:
1. 用户播放历史偏好（如果存在）
2. 用户收藏风格（如果存在）
3. 当前天气 + 时间（默认）

---

### 4. "今日专属"状态标记

**UI 元素**:
- 播放列表顶部显示"今日专属"标签（小型 pill badge）
- 标签样式：`background: linear-gradient(135deg, rgba(0,245,255,0.2), rgba(168,85,247,0.2))`

**显示时机**:
- 每次用户进入 app 时
- 推荐播放完成后刷新显示

---

### 5. 提示消息

**播放完成后的 Toast**:
```typescript
{
  message: "今日推荐已更新 🎵",
  type: "success",
  duration: 3000
}
```

---

## 与 A 阶段的关系

**A 阶段**: 首次用户体验 - AI 打招呼 + 推荐歌曲
**B 阶段**: 用户留存 - "欢迎回来" + 专属感

**复用逻辑**:
- `playWelcomeMessage()` 函数可复用
- 只需修改 TTS 文案
- 新增播放完成后的状态更新
